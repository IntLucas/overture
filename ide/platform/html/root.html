Overture Tools for Formal Modelling in VDM

The Overture Platform

Overture is an open source tool for the Vienna Development Method (VDM). The tool supports different VDM dialects which include the ISO VDM-SL standard, as well as the object-oriented extension VDM++ and the real-time extension of that called VDM-RT.

Getting Started

  • See the help containing: the VDM Language Manual, the User Guide and a collection of examples. Goto Help->Help Contents and look for Overture.
  • Documentation: Download user guides, tutorials, language manuals etc from the Overture SourceForge Documentation Download page
  • Visit the Overture website for general news: OvertureTool.org
  • Visit the Wiki for both language and tool specific FAQs: wiki.OvertureTool.org
  • New releases, examples, documentation and source are available at: Source Forge

Upgrading workspaces from earlier versions

The new Overture IDE is not compatible with workspaces from earlier versions, such as 0.2.x and earlier. Follow these steps to move your projects to the new IDE:

  1. Select a new empty workspace (Not an existing one used by Overture IDE 0.2.x)
  2. Select Import... -> Existing Projects into Workspace -> Next
  3. Select browse... next to "Select root directory" and navigate to your old workspace
  4. Check the projects which should be moved
  5. Tick the "Copy projects into workspace" box
  6. Click Finish and the projects will be copied into the new workspace
For more info see the Overture IDE online FAQ
