Overture Tools for Formal Modelling in VDM

The Overture Platform

Overture is an open tool platform build for the formal method called Vienna Development Method (VDM).

The platform supports different dialects which include the ISO VDM-SL standard as well as the object-oriented extension VDM++ and finally the real-time extension of that called VDM-RT.
