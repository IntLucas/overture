Overture Tools for Formal Modeling in VDM

The Overture Platform

Overture is an open tool platform build for the formal method called Vienna Development Method (VDM).

The platform supports different dialects which include the ISO VDM-SL standard as well as the object-oriented extension VDM++ and finally the real-time extension of that called VDM-RT.

Upgrading workspace from earlier version

The new Overture IDE is not compatible with workspaces from earlier versions such as 0.2.x and earlier. Follow the steps below to move your projects into the new IDE:
  1. Select a new empty workspace (Not an existing one used by Overture IDE 0.2.x)
  2. Select Import... -> Existing Projects into Workspace -> Next
  3. Select browse... next to "Select root directory" and navigate to your old workspace
  4. Check the projects which should be moved
  5. Press finish and the projects will be copied into the new workspace
For more info see the Overture IDE online FAQ
